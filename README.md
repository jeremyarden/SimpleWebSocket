## How to use

1. Pastikan Python 3 telah ter-install di komputer atau laptop Anda
2. Buka folder src pada file zip yang telah diunduh
3. Buka terminal sesuai dengan folder yang berisi file server.py
4. Pastikan PORT yang akan digunakan belum terpakai oleh proses lain
5. Ketikkan command untuk menjalankan server

``python3 server.py``
